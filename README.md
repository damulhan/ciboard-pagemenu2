# CIBoard-Pagemenu2

씨아이보드(CIBoard)는 코드이그나이터 기반 공개형 게시판입니다. 

Pagemenu2는 CIBoard의 기본 관리자 메뉴 기능을 확장한 메뉴 편집기입니다. 

기존 메뉴보다 좀 더 편리한 인터페이스를 메뉴를 편집할 수 있도록 했습니다. 

### 주요 기능 

- 다단 메뉴 
- 마우스로 순서를 자유스럽게 변환가능 
- 화면 변경없이 서버에 반영
